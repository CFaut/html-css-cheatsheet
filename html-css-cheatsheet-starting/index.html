Tables

Table Tags

Tag Name What it does
<table> Table The element representing all html tables
<thead> Table Head The element representing the column headers of a table
<tbody> Table Body Represents the rows that contain the data
<tr> Table Row Represents table row container
<td> Table Data Represents the table data cell
<tfoot> Table Footer The set of rows at the bottom of the table

Table Attributes

Attribute Name Description
colspan Column Span Represents how many columns a data element should span
rowspan Row Span Represents how many rows a data element should span
